# Shopping Screen Dashboard

> [!NOTE]
> 说明：这只是个示范 demo (This is just a demonstration demo).

A high-fidelity data visualization dashboard for shopping analytics. 

## 🎨 Design Theme
The dashboard features a sophisticated "deep navy and amber" theme. It transitions from traditional high-saturation cyan/blue aesthetics to a premium, low-saturation dark blue palette accented with warm orange and amber. It incorporates advanced visual effects including depth enhancement, bloom, and glass-like materials (glassmorphism) across both 2D and 3D map views.

## 🚀 Tech Stack
- **Core**: HTML5, Vanilla JavaScript, CSS3
- **Build Tool**: [Vite](https://vitejs.dev/) (v8+)
- **Visualization**: [ECharts](https://echarts.apache.org/) & ECharts-GL for rich data charting and 3D map visualizations.

## 📁 Project Structure
- `index.html`: Main entry point and layout structure.
- `src/css/`: Styling files including layouts, components, and global themes.
- `src/js/`: Logic files including ECharts configurations and interactivity.
- `src/main.js`: Main application script tying everything together.
- `public/`: Static assets.

## 📦 Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation

1. Clone the repository
2. Install dependencies:
```bash
npm install
```

### Running Locally
To start the Vite development server:
```bash
npm run dev
```

### Building for Production
To build the application for production:
```bash
npm run build
```
The optimized files will be generated in the `dist` directory.

### Preview Production Build
To preview the generated production build locally:
```bash
npm run preview
```

# 购物大屏数据看板 (Shopping Screen Dashboard)

> [!NOTE]
> 说明：这只是个示范 demo (This is just a demonstration demo)。

一个用于购物数据分析的高保真数据可视化看板。

## 🎨 设计主题
本看板采用精致的高级感“深藏青与琥珀金”主题。它摒弃了传统的高饱和度青蓝色调，转而使用更具质感的低饱和度深蓝色系，并辅以温暖的橙色和琥珀色作为点缀。在 2D 和 3D 地图视图中，广泛运用了景深增强、发光（Bloom）以及玻璃拟态（Glassmorphism）等高级视觉效果。

## 🚀 技术栈
- **核心**: HTML5, 原生 JavaScript (Vanilla JS), CSS3
- **构建工具**: [Vite](https://vitejs.dev/) (v8+)
- **数据可视化**: [ECharts](https://echarts.apache.org/) & ECharts-GL，用于丰富的图表呈现与 3D 地图可视化。

## 📁 项目结构
- `index.html`: 主入口文件及页面布局结构。
- `src/css/`: 样式文件目录，包含布局、组件和全局主题样式。
- `src/js/`: 逻辑处理目录，包含 ECharts 配置及各类交互脚本。
- `src/main.js`: 核心应用脚本，负责整体的调度与初始化。
- `public/`: 静态资源目录。

## 📦 快速开始

### 环境要求
请确保您的计算机上已安装 Node.js。

### 安装依赖

1. 克隆本项目
2. 安装所需依赖：
```bash
npm install
```

### 本地开发
启动 Vite 本地开发服务器：
```bash
npm run dev
```

### 生产环境构建
将应用打包为生产环境版本：
```bash
npm run build
```
优化后的文件将被生成在 `dist` 目录中。

### 预览生产环境构建
在本地预览生成的生产环境代码：
```bash
npm run preview
```
